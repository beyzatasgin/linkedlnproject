# Ekran Görüntüleri

Bu klasöre projenin ekran görüntüleri eklenecektir.

## Eklenmesi Gereken Görüntüler

1. **Ana Sayfa (Feed)** - `feed.png`
   - Gönderi listesi
   - Gönderi oluşturma formu

2. **Giriş Sayfası** - `login.png`
   - Giriş formu
   - Beni Hatırla checkbox'ı

3. **Kayıt Sayfası** - `register.png`
   - Kayıt formu

4. **Profil Sayfası** - `profile.png`
   - Kullanıcı profili görünümü

5. **Profil Düzenleme** - `profile-edit.png`
   - Profil düzenleme formu

6. **Bağlantılar Sayfası** - `connections.png`
   - Gelen/gönderilen istekler
   - Bağlantı listesi

7. **Diğer Kullanıcı Profili** - `user-profile.png`
   - Başka bir kullanıcının profili

## Görüntü Formatı

- Format: PNG veya JPG
- Çözünürlük: Minimum 1920x1080 (tercihen)
- Dosya boyutu: Mümkün olduğunca optimize edilmiş
